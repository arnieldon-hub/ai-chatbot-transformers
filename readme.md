\# AI Chatbot with Hugging Face Transformers\
\
This project demonstrates a simple conversational AI chatbot built using
the Hugging Face Transformers library and BlenderBot.\
\
\## Model Used\
facebook/blenderbot-400M-distill\
\
\## Technologies\
- Python\
- Hugging Face Transformers\
- PyTorch\
\
\## Features\
- Maintains conversation history\
- Generates contextual responses\
- Uses pretrained BlenderBot model\
\
\## Run the chatbot\
\
Install dependencies:\
\
pip install transformers torch\
\
Run the chatbot:\
\
python chatbot.py\
\
\## Example Conversation\
\
User: Hello, how are you?\
\
Bot: I\'m doing well. I enjoy playing video games and watching movies.
